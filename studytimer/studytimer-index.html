⏰ StudyTimer

과학적 학습 방법론을 기반으로 한 스마트 타이머 앱

📱 앱 소개

StudyTimer는 뽀모도로, Flowtime, 52/17 규칙 등 5가지 과학적 학습 모드를 제공하는 Android 타이머 앱입니다. 부모-자녀 연결 기능을 통해 학습 진도를 실시간으로 모니터링할 수 있으며, 고급 통계 분석으로 효과적인 학습 습관을 만들어갑니다.

🎯

5가지 학습 모드

뽀모도로, Flowtime, 52/17 규칙, 울트라디언 리듬, 커스텀 모드

👨‍👩‍👧‍👦

부모-자녀 연결

초대 코드로 간편 연결, 실시간 학습 모니터링

📊

고급 통계 분석

모드별 성과 추적, 학습 패턴 분석, 개인화 추천

🔔

스마트 알림

실시간 푸시 알림, 목표 달성 알림, 응원 메시지

🎨 학습 모드

🍅 뽀모도로

25분 집중 + 5분 휴식
4사이클 후 긴 휴식

🌊 Flowtime

자유 집중 시간
자동 휴식 계산 (1/5 비율)

⏰ 52/17 규칙

52분 집중 + 17분 휴식
DeskTime 연구 기반

🔄 울트라디언 리듬

90분 집중 + 20분 휴식
생체 리듬 기반

🎨 커스텀

1-90분 학습
1-30분 휴식 자유 설정

📋 앱 사양

  • 버전: 1.1
  • 플랫폼: Android
  • 최소 버전: Android 7.0 (API 24) 이상
  • 권장 버전: Android 14 (API 36) 이상
  • 언어 지원: 한국어, 영어, 일본어, 중국어
  • 가격: 무료 (프리미엄 기능 앱 내 결제)
  • 크기: 약 15MB
